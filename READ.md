# YouTube Clone - MiniTube

A responsive YouTube clone built using React and Tailwind CSS.

## Features

- Home page with video thumbnails
- Like & Watch Later functionality
- Shorts page
- Sidebar navigation
- Search bar
- Footer with social links

## Getting Started

```bash
npm install
npm start
